# Portfolio Personal — León Serrano

Portfolio profesional desarrollado con React, TypeScript y Tailwind CSS.

## 🛠️ Stack

- React + TypeScript
- Tailwind CSS v4
- Framer Motion
- Vite
- Deploy en Vercel

## 🚀 Demo

https://portfolio-nine-rho-6scaw4hmj5.vercel.app/

## 📁 Estructura
src/
├── components/     # Componentes reutilizables (Navbar, Footer, ProyectoCard)
├── sections/       # Secciones del portfolio (Hero, SobreMi, Habilidades, Proyectos, Contacto)
├── data/           # Datos estáticos (proyectos, habilidades)
└── utils/          # Utilidades compartidas (animaciones)


## 🏃 Correr localmente

```bash
npm install
npm run dev
```
